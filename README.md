# Proxy Tools Suite

Комплект инструментов для загрузки и проверки прокси-серверов.

## Описание

Этот проект предоставляет набор скриптов для:
- Загрузки списков бесплатных прокси из множественных источников
- Проверки работоспособности прокси с поддержкой многопоточности
- Сортировки прокси по протоколам (HTTP, HTTPS, SOCKS4, SOCKS5)
- Географической фильтрации (GeoIP)

## Структура проекта

```
├── load_proxy_list_new.py      # Загрузчик списков прокси
├── proxycheckerasync_add.py    # Асинхронный проверщик прокси
├── proxycheckerasync_my_new_3.py  # Альтернативный проверщик прокси
├── URLs.json                   # Конфигурация источников прокси
├── first_run_for_debian.sh     # Скрипт первичной настройки для Debian/Ubuntu
├── a.sh                        # Вспомогательный скрипт
└── README.md                   # Этот файл
```

## Установка

### Быстрая установка (Debian/Ubuntu)

```bash
chmod +x first_run_for_debian.sh
./first_run_for_debian.sh
pip install aiohttp requests beautifulsoup4 rich termcolor
```

### Ручная установка

1. Установите зависимости:
```bash
pip install aiohttp requests beautifulsoup4 rich termcolor
```

2. При необходимости установите опциональные пакеты:
```bash
pip install pyfreeproxies
```

## Использование

### 1. Загрузка прокси-листов

```bash
python3 load_proxy_list_new.py
```

Скрипт загружает списки прокси из источников, указанных в `URLs.json`, и сохраняет их в директорию `proxy_in/`.

### 2. Проверка прокси

```bash
python3 proxycheckerasync_add.py
```

Или альтернативный проверщик:

```bash
python3 proxycheckerasync_my_new_3.py
```

Проверщик:
- Читает прокси из директории `proxy_in/`
- Проверяет доступность каждого прокси
- Сохраняет рабочие прокси в директорию `proxy_out/`

## Конфигурация

### URLs.json

Файл содержит список источников прокси, организованных по протоколам:

```json
{
  "http": [
    {
      "url": "https://example.com/proxy-list.txt",
      "timestamp": "2025-01-01 00:00:00",
      "status": "active"
    }
  ],
  "https": [...],
  "socks4": [...],
  "socks5": [...]
}
```

### Настройки проверщика (proxycheckerasync_add.py)

- `CHECK_TIMEOUT` - таймаут проверки одного прокси (секунды)
- `MAX_CONCURRENT_CHECKS` - максимальное количество одновременных проверок
- `TEST_URLS` - список URL для тестирования прокси

## Директории

- `proxy_in/` - входная директория для сырых списков прокси
- `proxy_out/` - выходная директория для проверенных рабочих прокси

## Особенности

- **Асинхронная проверка** - высокая скорость благодаря asyncio
- **Множественные тестовые URL** - разнообразие источников для проверки
- **Богатый вывод** - использование Rich для красивого отображения прогресса
- **Гео-фильтрация** - поддержка GeoIP для фильтрации по странам
- **Детальная статистика** - отчёт о результатах проверки
